# 🍔 Restaurant Management System in C++

A simple console-based **Restaurant Management System** made in C++.

## Features
- Create new orders (Pizza, Burger, Pasta, Salad)
- View all orders
- Update order quantity
- Delete orders
- View total sales in PKR
- Data saved in `orders.txt` file (persists after closing program)

## How to Run
1. Download or clone the repository
2. Open `resturantmanagment830.cpp` in any C++ compiler (Dev-C++, Code::Blocks, Visual Studio, etc.)
3. Compile and run the program
4. Enjoy managing the restaurant! 

## Technologies
- C++
- File Handling
- Windows Console (system("cls"), getch())
